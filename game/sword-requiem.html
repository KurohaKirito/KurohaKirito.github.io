魔剑镇魂曲正式版新增内容

发表于 2025-06-03 更新于 2025-06-04 分类于 Game 本文字数： 818 阅读时长 ≈ 3 分钟
魔剑镇魂曲正式版新增内容

🌀 技能重构与机制打磨，让每次出手更具重量

技能系统经历了全面调整，不仅效果表现更直观，机制逻辑也更加清晰。许多技能获得了重做或优化，使其在实战中的作用更加鲜明，战斗过程更具张力。

🔥 九项全新技能，扩展你的战术可能

新增技能覆盖多个维度：爆发输出、生存反击、战场控制与协作支持，适配不同战斗风格，也为角色成长与构筑提供更多方向。

🎯 自由成长系统，随时重塑角色定位，不能反悔？不存在的！

角色养成不再拘泥于预设路线。无论是职业搭配还是技能组合，都可依据你的理解与喜好自由选择，并且支持重置与调整，无任何试错成本，尽情尝试。

🕹️ 四种模式可选，自定义玩法由你决定

游戏现支持四种游玩模式，从剧情体验到挑战竞分应有尽有。特别是自定义模式，允许你修改敌我数值、规则参数，甚至添加独特的战斗彩蛋，打造属于你的专属对局, 还有致敬彩蛋哦！

🏹圣器系列登场，每一把都是独特解法

圣剑、圣斧、圣杖等武器加入战局，各自拥有独立机制与玩法倾向，为战斗带来额外变量，等待你去发掘和掌控。

📖 全新 IF 剧情，来看克何林和夏侬如何改变自身命运

觉得克何林、夏侬、夏绿蒂等人很可惜，新的 if 剧情满足你，来看主人公的不同选择让角色的命运发生了什么变化！

🛠️ 内置控制台上线，测试与娱乐两相宜，甚至让敌方首领加入我方

内置控制台主要用于模组制作者们测试, 当然你也可以用来爽玩！ 可以随意提升等级, 添加想要的物品, 哪怕是正常途径无法获得的物品！ 甚至还可以任意改变阵营, 让神将哈萨加入我方会怎样呢 ? 这一切就看你的了 !

🌟 启示与觉醒：双机制改变战局节奏

启示机制可使待机角色再次行动，用于连锁战术或紧急响应。觉醒机制则让技能发生质变，获得全新效果或追加能力，赋予单个单位压制战局的可能性。

🧩 创意工坊全面支持，自由构建你想要的世界

游戏已集成模组管理与上传功能，无需离开游戏即可订阅、启用或制作模组。支持内容覆盖角色、剧情、地图、音效、技能逻辑等，并支持 C# 编写自定义技能逻辑。示例模组已内置，供开发者参考与改写。

🏳‍🌈 排行榜

困难与挑战模式下支持上传最高分纪录，看看谁才是真正的战术大师。
